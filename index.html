how to make a leaderstats script in rblx studio

open roblox studio

Then click new experience

it shold now look like this

then hover over SeverscriptService

and cLick the plus icon and search script and select it

now type

game.Players.PlayerAdded:Connect(function(player)

local ls = Instance.new("Folder", player)

ls.Name = "leaderstats"

local coins = Instance.new("IntValue", ls)

coins.Name = "Coins"

coins.Value = 10

end)

click play and then

it should look like this now
